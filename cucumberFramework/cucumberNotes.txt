//7. Cucumber tags & Background keyword 
       >> We can sepearte our test cases marking tags in Feature File like
       
           @regression
           Scenario: Testing Software with Regression
           
           @smoke
           Scenario: Testing Software with Smoke
           
           and Providing tags in Cucumber Options along with Feature, Glue in Runner File  like
           
            @CucumberOptions
            (
		       features = "src/test/java/featurePackage/",
		       glue = "Step_definitions",
		       dryRun = true,
		       tags = "@smokeTest and not @regressionTest",
		       
		       
		       
    Backgorund Keyword:  
          Repeated Steps in each test scnearion in a feature file, can defined user using Background.
           and will execute BEFORE EVERY Scenario.
           
          Ex.-  
          Background: 
          Given Setup database enteries
          When browser is launched from config variables
          And hit the home page url of banking site
          
          
          Scenario:
          Given Admin page launched........etc
          
          

//10. Cucumber Hooks
         -- Similar to Background Keyword
         -- Used to Starting a Browser, Clearing Cookies, Connecting Database.....etc
         -- Some Hooks: 
                > @Before, i.e. Before Each Scenario
                > @After, i.e. After Each Scenario
                > @Before(order=1)
                > @Before(order=2, value="@screenshots") i.e., Execution based on order and tags
                > @BeforeStep
                > @AfterStep
                
         -- Order of Hooks:
              Before > Background > Scenario/Scneario Outline > After
